WTF is this?

For now is just a alpha and highly biased easter egg.

Why?

I wanted to read something like this - an information flood is drowning our times - but I couldn’t find it elsewhere. So, as most projects, it started while «scratching my own itch» and before becoming a website it lived as a markdown file.

Making sense of chaos is a sisyphean task but I see it as a way of using my procrastination positively.

How do you curate it?

I cherry-pick relevant stuff and list them carefully. It’s certainly affected by my biases (I’m aware of these) but links are not sponsored nor will ever be.

How often is Underrated section updated?

Only when relevant. Visit every month and «you’re good to go». New listings are easy to spot.

Can I contribute?

Open an issue or pull request the repository with relevant links, corrections or suggestions. I’m curating only a few pages but I accept pull requests creating/curating more - if you can handle the burden of using raw HTML. For now, this is my filter to accept them.

Feeling grateful?

Shop something through my Amazon affiliate link (you pay exactly the same but it keeps my motivation high) or pay what you think is fair. I don’t want to publish intrusive ads neither data farm with yet another stupid login.

Colophon

Hand coded - fully functional without JS - and statically hosted at Netlify via a Github repository. Tufte CSS (adapted) with native darkmode. No images, no tracking (GDPR friendly analytics with Goatcounter) and no bloat. Ok, it uses non-standard fonts but web design is mostly typography.
